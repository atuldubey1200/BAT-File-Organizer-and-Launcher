# BAT-File-Organizer-and-Launcher
A GUI dashboard for organizing, managing, and launching BAT files with searchable metadata, developer information, and JSON-based project collections.

## Features

- Launch BAT files directly from the dashboard
- Organize applications with custom names
- Store descriptions and developer information
- Search and filter applications
- JSON-based autosave system
- Import and merge launcher collections
- Track last-used timestamps
- Open application folders directly

## Requirements

- Python 3.9+

## Installation

```bash
pip install -r requirements.txt
```

Run:

```bash
python BAT_Launcher_Dashboard.py
```
